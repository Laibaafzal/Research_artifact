# License and Data Usage

This repository is open-source.

The raw subtitle data was originally sourced from OpenSubtitles.
Which is publicly accessible. Users are free to download subtitle files from OpenSubtitles themselves. In this project, the data has been cleaned, processed, and manually labeled by the author.

Important notes:

The dataset released in this repository does not redistribute the original raw subtitle files verbatim. Only manually labeled and processed data is included. Therefore, there is no copyright issue with releasing this dataset.

The dataset does not contain any personally identifiable information (PII) or sensitive personal data. All content is derived from fictional characters and dialogues from movies.
